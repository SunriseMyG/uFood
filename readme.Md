## 🎯 About

**uFood** is a modern, responsive web application that allows users to discover restaurants, view detailed information, explore locations on interactive maps, and manage their favorite dining places. Built with cutting-edge technologies, uFood delivers a seamless and intuitive user experience.

---

## ✨ Features

- 🔍 **Restaurant Discovery** - Browse and search through a wide variety of restaurants
- ❤️ **Favorites Management** - Save your favorite restaurants for quick access
- 🗺️ **Interactive Maps** - View restaurant locations and get routing directions
- 👤 **User Profiles** - Create and manage your user account
- 🔐 **Authentication** - Secure login system to protect your preferences
- 📱 **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- 🎠 **Restaurant Carousel** - Browse featured restaurants in an elegant carousel
- ⚡ **Fast Performance** - Lightning-fast loading and smooth interactions

## 🛠️ Tech Stack

### Frontend Framework & Build
- **[Preact](https://preactjs.com/)** - Fast and lightweight React alternative
- **[React Router DOM](https://reactrouter.com/)** - Client-side routing
- **[Vite](https://vitejs.dev/)** - Next-generation frontend build tool
- **[TypeScript](https://www.typescriptlang.org/)** - Static type checking for JavaScript

### Utilities & Libraries
- **[Axios](https://axios-http.com/)** - Promise-based HTTP client for API requests
- **[Leaflet](https://leafletjs.com/)** - Interactive map library
- **[Leaflet Routing Machine](http://www.liedman.net/leaflet-routing-machine/)** - Routing engine for maps
- **[React Icons](https://react-icons.github.io/react-icons/)** - Beautiful icon library
- **[React Slick](https://react-slick.neostack.com/)** - Carousel component

## 📦 Installation

### Prerequisites
- **Node.js** (v16 or higher)
- **npm** or **yarn** package manager

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/SunriseMyG/uFood.git
   cd uFood
   ```

2. **Navigate to the frontend directory**
   ```bash
   cd front
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Set up environment variables** (if needed)
   ```bash
   cp .env.example .env
   ```
   Edit `.env` file with your configuration

## 🚀 Getting Started

### Development Mode

Start the development server with hot module replacement:

```bash
npm run dev
```

The application will be available at **[http://localhost:5173](http://localhost:5173)**

### Production Build

Build the project for production:

```bash
npm run build
```

### Preview Production Build

Preview the production build locally:

```bash
npm run preview
```
